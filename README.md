# media_viewer
Web based app for displaying holiday happy snaps that allows favorite images to be shown more often and different progression speeds for different purposes/locations. Being a web based app means that it can run on anything with a browser (including turning long outdated ipads into digital photo frames). Preloader keeps it progressing smoothly.

App gives full control of slide progression speed, filtering options, weighted pseudo-random ordering, and user based favourites.

Intended for deployment on a linux server on the local network - [deploy/README.md](deploy/README.md)
